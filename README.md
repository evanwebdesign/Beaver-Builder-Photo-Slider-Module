# Beaver Builder Photo Slider Module

A custom Beaver Builder module that displays a responsive photo slider with thumbnail navigation, autoplay controls, keyboard accessibility, and mobile swipe support.

This module was developed for use on WordPress websites utilizing Beaver Builder and is maintained as a reusable component for future projects.

## Features

- Responsive image slider
- Thumbnail navigation
- Previous/Next controls
- Optional autoplay
- Pause/Play controls
- Keyboard accessible
- Mobile swipe support
- Focus management improvements
- Respects reduced motion preferences
- Pauses when slider is not visible on screen

## Requirements

- WordPress
- Beaver Builder
- PHP 7.4+ (or version required by your hosting environment)

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress.
3. Ensure Beaver Builder is installed and activated.
4. The "Photo Slider" module will become available within Beaver Builder.

## Version

## Historical Notes

This repository represents the photo-only branch of the slider.

A separate Photo + Video Slider fork was later created that introduced support for YouTube and Vimeo embeds. The two projects should be treated as separate codebases to avoid reintroducing video-related complexity into the photo-only version.
